# Python - Async

This project covers asynchronous programming in Python using `asyncio`.

## Tasks

0. **The basics of async** - Write an asynchronous coroutine `wait_random` that waits for a random delay.
1. **Execute multiple coroutines** - Write `wait_n` to spawn multiple `wait_random` coroutines.
2. **Measure the runtime** - Write `measure_time` to measure the execution time of `wait_n`.
3. **Tasks** - Write `task_wait_random` that returns an `asyncio.Task`.
4. **Tasks** - Write `task_wait_n` using `task_wait_random`.
